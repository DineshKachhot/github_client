### github_client

# Introduction

Github Client is a mobile app developed in React Native (expo).

## Installation

Clone this repo in your local

Install dependancies

```bash
npx expo install
```

Run app

```bash
npx expo start
```

## Functionalities

- Expo project
- Typescript
- Axios
- TanStack Query
- Redux
- React Native Navigation
